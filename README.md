# Zurikheyzho

A conlang project and grammar site built with Hugo.

## Overview

This repository contains a website for presenting a conlang, built using the Hugo static site generator.

The site includes:

- Grammatical descriptions
- Example sentences with glosses
- A small core vocabulary

## Demo

https://zurikheyzho.com/

## Structure

    content/        # Linguistic content (CC BY-SA)
    static/         # Static assets (images, favicon, etc.)
    themes/         # Theme files

## Development

### Requirements

- Hugo (v0.152.2)

### Run locally

    hugo server

## Deployment

This site is deployed via Netlify:

- Connected to GitHub
- Automatically built and deployed on push
